## Installation

You can install the package via composer:

```bash
composer require astrogoat/equalweb
```

## Usage

```php
$equalweb = new Astrogoat\Equalweb();
echo $equalweb->echoPhrase('Hello, Astrogoat!');
```

## Testing

```bash
composer test
```
